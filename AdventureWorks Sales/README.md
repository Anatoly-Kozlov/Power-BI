# Adventure Works Sales Dashboard
## Обзор проекта

Данный проект представляет собой интерактивный аналитический дашборд в Power BI, построенный на основе датасета Adventure Works DW 2020.
Дашборд ориентирован на руководителей продаж и бизнес-пользователей и предназначен для мониторинга ключевых показателей, анализа динамики продаж и выявления точек роста.

Проект охватывает полный BI-цикл: от загрузки данных и моделирования до написания DAX-мер и построения визуализаций.

## Цели и бизнес-задачи

- Контроль выручки и прибыльности бизнеса
- Сравнение текущих показателей с предыдущим годом
- Анализ продаж по каналам, продуктам и географии
- Выявление наиболее прибыльных категорий и рынков
- Детальный анализ Internet Sales на уровне клиентов

## Источник данных

- Данные загружены из AdventureWorks Sales.xlsx


## Модель данных

В проекте реализована звёздная схема данных. Меры выненесены в отдельную таблицу Measure

В таблицу Date добавлены вычисляемые столбцы:
- Year - используется в фильтрах
- Quarter - используется в фильтрах
- Month Short - используется в фильтрах и визуализациях
- MonthN - используется для корректной сортировки месяцев
